# XPENSE — Expense Tracker

A sleek, dark-themed expense tracker that runs entirely in your browser. No server needed.

## Features

- ✅ Add income & expense transactions
- ✅ Categorize (Food, Transport, Bills, Health, etc.)
- ✅ Live summary: Total Income, Expenses, Net Balance
- ✅ Filter by type or category
- ✅ Category breakdown with visual bars
- ✅ Export all transactions to CSV
- ✅ Data persists in browser localStorage
- ✅ Fully responsive (mobile-friendly)

## Usage

1. Open `index.html` in any modern browser
2. Add transactions using the form
3. Use filters to view specific categories
4. Export your data via the "↓ Export CSV" button

## No installation required — pure HTML, CSS, and JavaScript.
