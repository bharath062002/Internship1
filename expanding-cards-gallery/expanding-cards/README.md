# Expanding Cards Gallery

An elegant, interactive image gallery with smooth expanding card animations.

## Features
- Click any card to expand it with a smooth animation
- Keyboard navigation (← → arrow keys)
- Dot indicators at the bottom
- Responsive layout (stacks vertically on mobile)
- Beautiful typography using Cormorant Garamond + Montserrat
- Grain texture overlay for depth

## Usage
Simply open `index.html` in any modern browser. No build step or dependencies required.

## Customization
- **Images**: Replace the Unsplash URLs in each `.card-bg` `style` attribute
- **Text**: Edit the `card-title`, `card-desc`, and `card-label` content
- **Colors**: Modify the CSS variables in `:root`
- **Cards**: Add or remove `.card` blocks (update `card-number` text accordingly)
